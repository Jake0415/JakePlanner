# Task 012: 사용자 관리 페이지 UI 구현

## 상태
- [ ] 미완료

## 개요
사용자 관리 페이지의 UI를 구현합니다. 사용자 목록 테이블, 사용자 추가/수정/삭제 모달,
역할별 권한 안내, 슈퍼어드민 전용 접근 제한 UI를 포함합니다.

## 관련 PRD 기능
- [F013] 사용자 관리

## 관련 파일
- `app/(auth)/users/page.tsx` — 사용자 관리 페이지
- `components/users/user-table.tsx` — 사용자 목록 테이블
- `components/users/user-form-modal.tsx` — 사용자 추가/수정 모달
- `components/users/role-info.tsx` — 역할별 권한 안내

## 수락 기준
- [ ] 사용자 목록 테이블이 더미 데이터로 표시됨
- [ ] 사용자 추가 모달 폼이 동작함
- [ ] 사용자 정보 수정/삭제 기능 UI가 동작함
- [ ] 역할별 권한 안내가 표시됨
- [ ] 슈퍼어드민 전용 접근 제한 UI가 처리됨

## 구현 단계
- [ ] 사용자 목록 테이블 구현 (이름, 이메일, 부서, 역할, 가입일)
- [ ] 사용자 추가 모달 폼 구현 (이메일/비밀번호/이름/전화번호/부서명, 역할 선택)
- [ ] 사용자 정보 수정 모달 구현
- [ ] 사용자 삭제 확인 다이얼로그 구현
- [ ] 역할별 권한 안내 표시 구현 (admin/member 차이 설명)
- [ ] 슈퍼어드민 전용 접근 제한 UI 처리

## 테스트 체크리스트
- [ ] 사용자 목록 테이블 더미 데이터 표시
- [ ] 사용자 추가/수정/삭제 모달 동작
- [ ] 비 슈퍼어드민 접근 시 제한 UI 표시

## 참고사항
더미 데이터 기반으로 UI만 구현하며, 실제 사용자 관리 API 연동은 Task 021에서 구현합니다.
