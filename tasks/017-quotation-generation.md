# Task 017: 3가지 견적안 자동 생성 구현 (F005)

## 상태
- [ ] 미완료

## 개요
RFP 파싱 결과를 기반으로 3가지 견적안(수익성 중심, 규격 충족, 성능 향상)을 자동 생성하는 엔진을 구현합니다.
부품 자동 매칭 로직과 OpenAI GPT-4o 연동을 통한 부품 추천 및 조합 최적화를 포함합니다.

## 관련 PRD 기능
- [F005] 견적 생성

## 관련 파일
- `app/api/quotation/generate/route.ts` — 3가지 견적안 생성 API
- `app/api/quotation/[id]/route.ts` — 견적 상세 조회 API
- `lib/quotation/matching-engine.ts` — 부품 매칭 엔진
- `lib/quotation/profitability.ts` — 수익성 중심안 알고리즘
- `lib/quotation/spec-match.ts` — 규격 충족안 알고리즘
- `lib/quotation/performance.ts` — 성능 향상안 알고리즘
- `lib/ai/part-recommender.ts` — AI 부품 추천

## 수락 기준
- [ ] RFP 파싱 결과 기반 부품 자동 매칭이 동작함
- [ ] 수익성 중심안이 마진 최대화 부품 조합으로 생성됨
- [ ] 규격 충족안이 RFP 요구사항 정확 매칭으로 생성됨
- [ ] 성능 향상안이 10~30% 업스펙 부품 조합으로 생성됨
- [ ] 견적 생성 페이지가 실제 API와 연동됨

## 구현 단계
- [ ] RFP 파싱 결과 기반 부품 자동 매칭 로직 구현
- [ ] 수익성 중심안 알고리즘 구현 (마진 최대화 부품 조합)
- [ ] 규격 충족안 알고리즘 구현 (RFP 요구사항 정확 매칭)
- [ ] 성능 향상안 알고리즘 구현 (10~30% 업스펙 부품 조합)
- [ ] `POST /api/quotation/generate` 3가지 견적안 생성 API 구현
- [ ] `GET /api/quotation/[id]` 견적 상세 조회 API 구현
- [ ] OpenAI GPT-4o 연동 - 부품 추천 및 조합 최적화
- [ ] 견적 생성 페이지 더미 데이터를 실제 API로 교체
- [ ] 견적 생성 로직 테스트
