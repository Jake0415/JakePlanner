# Task 007: 대시보드 페이지 UI 구현

## 상태
- [ ] 미완료

## 개요
대시보드 페이지의 레이아웃을 구현합니다. 상단 통계 카드, 최근 견적 목록 테이블, 빠른 액션 버튼 영역을 포함하며,
역할별 UI 분기 처리를 더미 역할 상태를 활용하여 구현합니다.

## 관련 PRD 기능
- [F012] 대시보드

## 관련 파일
- `app/(auth)/dashboard/page.tsx` — 대시보드 페이지
- `components/dashboard/stat-cards.tsx` — 통계 카드 영역
- `components/dashboard/recent-quotations.tsx` — 최근 견적 목록
- `components/dashboard/quick-actions.tsx` — 빠른 액션 버튼

## 수락 기준
- [ ] 상단 통계 카드(진행 중, 완료, 낙찰, 월간 낙찰률)가 표시됨
- [ ] 최근 견적 목록 테이블이 더미 데이터로 표시됨
- [ ] 빠른 액션 버튼이 역할에 따라 다르게 표시됨
- [ ] 역할별 UI 분기 처리가 동작함

## 구현 단계
- [ ] 대시보드 레이아웃 구조 구현
- [ ] 상단 통계 카드 구현 (진행 중 건수, 완료 건수, 낙찰 건수, 월간 낙찰률)
- [ ] 최근 견적 목록 테이블 구현 (최신 5건, 더미 데이터)
- [ ] 빠른 액션 버튼 영역 구현
- [ ] RFP 업로드 바로가기 버튼
- [ ] 부품 관리 바로가기 버튼 (관리자 이상만 표시)
- [ ] 사용자 관리 바로가기 버튼 (슈퍼어드민만 표시)
- [ ] 역할별 UI 분기 처리 구현 (더미 역할 상태 활용)

## 테스트 체크리스트
- [ ] 통계 카드 4개 정상 렌더링
- [ ] 최근 견적 테이블 더미 데이터 표시
- [ ] 역할별 빠른 액션 버튼 표시/숨김 동작
