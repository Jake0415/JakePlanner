# Task 008: 부품 관리 페이지 UI 구현

## 상태
- [ ] 미완료

## 개요
부품 관리 페이지의 전체 레이아웃을 구현합니다. 카테고리 탭 전환, 부품 목록 데이터 테이블,
부품/카테고리 추가·수정·삭제 모달 폼, 가격 4단계 입력과 마진 자동 계산을 포함합니다.

## 관련 PRD 기능
- [F001] 부품 관리
- [F002] 부품 가격 관리

## 관련 파일
- `app/(auth)/parts/page.tsx` — 부품 관리 페이지
- `components/parts/parts-table.tsx` — 부품 목록 테이블
- `components/parts/part-form-modal.tsx` — 부품 추가/수정 모달
- `components/parts/category-tabs.tsx` — 카테고리 탭/그룹 전환
- `components/parts/category-form-modal.tsx` — 카테고리 추가/삭제 모달

## 수락 기준
- [ ] 카테고리 탭(서버 부품 / 네트워크·인프라) 전환이 동작함
- [ ] 부품 목록 데이터 테이블이 검색, 필터, 페이지네이션을 지원함
- [ ] 부품 추가/수정 모달 폼이 동작함 (가격 4단계 입력 포함)
- [ ] 마진 자동 계산이 표시됨
- [ ] 카테고리 추가/삭제 UI가 동작함
- [ ] 멤버 역할 시 조회 전용 UI로 표시됨

## 구현 단계
- [ ] 카테고리 탭/그룹 전환 UI 구현 (서버 부품 / 네트워크·인프라)
- [ ] 부품 목록 데이터 테이블 구현 (검색, 카테고리 필터, 페이지네이션)
- [ ] 부품 추가 모달 폼 구현
- [ ] 모델명, 제조사, 주요 사양 입력 필드
- [ ] 가격 4단계 입력: 리스트가, 시장가, 원가, 공급가
- [ ] 마진 자동 계산 표시 (공급가 - 원가)
- [ ] 부품 수정 모달 폼 구현
- [ ] 부품 삭제 확인 다이얼로그 구현
- [ ] 카테고리 추가 폼 구현 (카테고리명, 그룹, 스펙 필드 정의)
- [ ] 카테고리 삭제 시 하위 부품 존재 여부 경고 구현
- [ ] 멤버 역할 시 조회 전용 UI 처리

## 테스트 체크리스트
- [ ] 카테고리 탭 전환 정상 동작
- [ ] 부품 추가/수정/삭제 모달 동작 확인
- [ ] 마진 자동 계산 값 정확성 확인
- [ ] 멤버 역할 시 편집 버튼 비활성화 확인

## 참고사항

Task 005의 데이터 테이블 컴포넌트를 재사용합니다.
