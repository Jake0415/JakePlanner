# Task 010: 견적 생성 페이지 UI 구현

## 상태
- [ ] 미완료

## 개요
견적 생성 페이지의 레이아웃을 구현합니다. 3가지 견적안(수익성 중심/규격 충족/성능 향상) 탭 UI,
부품 구성 테이블, 마진 시뮬레이션 UI(슬라이더, 역산), 견적안 비교 요약 카드를 포함합니다.

## 관련 PRD 기능
- [F005] 견적 생성
- [F006] 마진 시뮬레이션

## 관련 파일
- `app/(auth)/quotation/page.tsx` — 견적 생성 페이지
- `components/quotation/quotation-tabs.tsx` — 3가지 견적안 탭
- `components/quotation/parts-config-table.tsx` — 부품 구성 테이블
- `components/quotation/margin-simulator.tsx` — 마진 시뮬레이션
- `components/quotation/comparison-summary.tsx` — 견적안 비교 요약

## 수락 기준
- [ ] 3가지 견적안 탭 전환이 동작함
- [ ] 각 안별 부품 구성 테이블이 더미 데이터로 표시됨
- [ ] 마진 시뮬레이션 슬라이더가 실시간 값 변경을 지원함
- [ ] 목표 마진 역산 기능 UI가 동작함
- [ ] 견적안 간 총액/마진 비교 요약 카드가 표시됨
- [ ] 견적 확정 버튼이 표시됨

- [ ] 견적안 간 총액/마진 비교 요약 카드 구현
- [ ] 견적 확정 버튼 구현
