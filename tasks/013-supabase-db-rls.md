# Task 013: Supabase 데이터베이스 구축 및 RLS 적용

## 상태
- [ ] 미완료

## 개요
Supabase 프로젝트에 마이그레이션 SQL을 실행하여 9개 테이블을 생성하고, RLS 정책을 적용합니다.
