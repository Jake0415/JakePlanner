# Task 006: 초기 설정 및 로그인 페이지 UI 구현

## 상태
- [ ] 미완료

## 개요
초기 설정 페이지의 스텝 폼 구조와 로그인 페이지의 UI를 구현합니다.
React Hook Form + Zod 연동을 통한 입력 유효성 검사와 반응형 레이아웃을 포함합니다.

## 관련 PRD 기능
- [F010] 인증 시스템
- [F011] 초기 설정

## 관련 파일
- `app/setup/page.tsx` — 초기 설정 페이지
- `components/setup/setup-form.tsx` — 초기 설정 폼 컴포넌트
- `app/login/page.tsx` — 로그인 페이지
- `components/login/login-form.tsx` — 로그인 폼 컴포넌트

## 수락 기준
- [ ] 초기 설정 스텝 폼이 Step 1 → Step 2 → 완료 순서로 동작함
- [ ] 로그인 폼이 이메일/비밀번호 입력 및 에러 표시를 지원함
- [ ] 입력 유효성 검사가 Zod 스키마와 연동됨
- [ ] 모바일/태블릿/데스크톱 반응형 레이아웃이 적용됨

## 구현 단계
- [ ] 초기 설정 페이지 스텝 폼 구조 구현 (Step 1 → Step 2 → 완료)
- [ ] Step 1: 슈퍼어드민 계정 정보 입력 폼 (이메일/비밀번호/이름/전화번호/부서명)
- [ ] Step 2: 회사 정보 입력 폼 (회사명/사업자등록번호)
- [ ] 입력 유효성 검사 UI 구현 (React Hook Form + Zod 연동)
- [ ] 성공/실패 피드백 UI 구현
- [ ] 로그인 페이지 이메일/비밀번호 입력 폼 구현
- [ ] 로그인 버튼 및 로딩 상태 구현
- [ ] 에러 메시지 표시 영역 구현
- [ ] 반응형 레이아웃 적용 (모바일/태블릿/데스크톱)

## 테스트 체크리스트
- [ ] 초기 설정 스텝 전환 정상 동작
- [ ] 필수 입력값 누락 시 유효성 검사 메시지 표시
- [ ] 로그인 폼 렌더링 및 입력 동작 확인
- [ ] 반응형 레이아웃 브레이크포인트별 확인
